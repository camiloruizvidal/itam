Esta es una breve documentacion del sistema.
El sistema esta desarrollado en php, usando el framework de adodb y adiciones de oderlogica.
Este sistema funciona con apache. Si desea usar otro servidor por favor verifique el redireccionamiento.
En primer lugar. Las plantillas deben usar un conjunto de etiquetas con esta denominacion <#--nombre_etiqueta--#>
Se trabaja con 4 etiquetas basicas en la plantilla.
<#--titulo--#> Es el title del head en html
<#--js--#> Aqui se cargan todas las librerias de javascript. solo deben especificar la ruta y ya.
<#--css--#> Aqui se cargan todas las librerias de css. solo deben especificar la ruta y ya.
<#--contenido--#> Aqui es donde se muestra el contenido que se desarrolla.
